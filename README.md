# Little Lemon Restaurant

A responsive restaurant website built with React and Vite. The project includes a reservation form with date-based availability, automated tests and a production-ready build setup.

## Features

* Responsive layout for desktop, tablet and mobile screens
* Restaurant landing page with navigation, hero section and menu content
* Reservation form with dynamically generated available times
* Time selection disabled until a reservation date is selected
* Form validation and user feedback
* Automated testing with Vitest and React Testing Library
* ESLint configuration for code-quality checks
* Optimized WebP images for improved loading performance

## Technologies

* React
* Vite
* JavaScript
* CSS
* Vitest
* React Testing Library
* ESLint

## Getting Started

Clone the repository and install the dependencies:

```bash
npm install
```

Start the local development server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```

Run the automated tests:

```bash
npm test -- --run
```

Run the ESLint checks:

```bash
npm run lint
```

## Testing

The project currently includes 13 automated tests covering:

* Date-based reservation availability
* Disabled and enabled states of the time-selection field
* Reservation form behavior
* Available-time generation logic
